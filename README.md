# Glazenwasser Demo Website

Dit repository bevat een **custom gemaakte demo / showcase website** voor een glazenwasser.
De code en het ontwerp zijn bedoeld **uitsluitend ter inzage en presentatie**.

## Doel van dit project
- Visual demo van een mogelijke website-opzet
- Ontwerp- en stijlvoorbeeld
- Interne of klantgerichte presentatie

## Gebruik & beperkingen
Deze repository is **NIET open source**.

Wat wel mag:
- Bekijken van de code

Wat NIET mag:
- Kopiëren of hergebruiken van (delen van) de code
- Deployen voor eigen of commerciële websites
- Aanpassen en opnieuw publiceren
- Doorverkopen of doorgeven aan derden

Alle rechten op ontwerp, structuur, tekst en styling blijven bij de auteur.
